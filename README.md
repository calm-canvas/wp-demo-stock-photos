# wp-demo-stock-photos
High-quality, royalty-free stock photos curated for WordPress theme and template demonstration. Use these open-source visuals to seed your WP demo sites, mockups, and presentations. All images are legally free for commercial and personal use (CC0/Public Domain).

## Sources

### Gratisography
Images in the `gratisograph/` folder are sourced from [Gratisography](https://gratisography.com/), a collection of free, high-resolution pictures licensed under the Creative Commons Zero (CC0) license. These images are free for both commercial and personal use.
